Q1

Given the point (x, y) = (3, -2), compute the network output ŷ.

With weights 2 (bias) and 1 (x₁): z = 2·1 + 1·x = 2 + x = 2 + 3 = 5, so ŷ = 5.

Q2

Given the point (x₁, x₂, y) = (2, -1, 3), compute the network output ŷ.

Weights: 1 (bias), 2 (x₁), −1 (x₂). z = 1·1 + 2·x₁ + (−1)·x₂ = 1 + 2·2 + (−1)·(−1) = 1 + 4 + 1 = 6, so ŷ = 6.

Q3

Given the point (x₁, y) = (3, 0), compute the sigmoid output ŷ.

Weights: 2 (bias), −1 (x₁). z = 2·1 + (−1)·x₁. For x₁ = 3: z = 2·1 + (−1)·3. ŷ = σ(2·1 + (−1)·3) = 1/(1 + e^{-(2·1 + (−1)·3)}).

Q5

Which of the preceding problems represents multiple regression?

Answer: 2. Q2 uses two independent input features (x₁, x₂) to predict y, which is multiple regression. Q1 has a single feature; Q3 has a single feature with a sigmoid (classification style); Q4 is a multi-layer network.

Q6

Which of the preceding problems represents logistic regression?

Answer: 3. Q3 has a single input feature and a sigmoid output, matching logistic regression. Q1 is simple linear (no sigmoid). Q2 is multiple linear regression (two input features). Q4 is a deeper network with hidden ReLU layer.

Q7

Which of the preceding problems represents simple linear regression?

Answer: 1. Q1 has a single input feature with a linear output (no hidden layer, no nonlinear activation), which is simple linear regression. Q2 adds an additional input feature (multiple regression). Q3 adds a sigmoid (logistic). Q4 introduces a hidden layer.
